Elementos block

↑
↑

Elementos block começam em uma nova linha, ocupam toda a largura disponível e vão se empilhando um em cima do outro conforme são inseridos, como os parágrafos dessa apresentação. Eles podem ser parágrafos, divs, sections, navs, headers, artciles e outros.

Elementos block também podem estar inseridos dentro de outro block:

Elementos inline

↑
↑

Elementos inline não começam em novo linha. Eles seguem o fluxo normal e vão se alinhando um ao lado do outro ocupando apenas a largura do seu próprio conteúdo. Esses elementos podem estar um dentro do outro e dentro de um elemento block.

lComumente usamos inline para destacar algumas palavras com sublinhados, negritos, itálicos ou com cores diferentes.

Semântica

↑
↑

Semântica, no HTML, é a prática de dar significado e estrutura aos elementos da página. Existem muitos benefícios nesse uso, incluindo habilitar devices a lerem seu código da forma mais adequada, por exemplo, um leitor de feed, pode pegar o seu primeiro <h1> e o seu primeiro <p> para montar um preview do conteúdo.

Além disso, HTML semântico facilita a leitura do código e possíveis retrabalhos.

↑
↑

<div> e <span>

↑
↑

Divs e Spans são dois elementos essenciais para qualquer código HTML mas, ao contrário dos outros, eles não tem nenhum valor semântico, então, servem apenas como containers para serem estilizados.

Por exemplo, parágrafos são colocado dentro da tag <p> e títulos são colocados nas tags <h1> até <h6> por serem headings.

Divs são elementos block e Spans são elementos inline.

↑
↑

Classes e ids

↑
↑

É normal ver divs e spans com atributos de classe e id. Esses atributos servem pra "nomear" esses elementos para que eles possam ser alvos de links e estilos.


Classes: A mesma classe pode ser aplicada em mais de um elemento.

<a class="menu"> Movile </a>
<span class="menu"> Movile </span>

Tanto a tag <a> quanto a <span> tem a classe menu


Ids: também podem ser usados para estilizar, mas apenas 1 elemento.

<div id="intro"> Introdução da página </div>
<div id="about"> Sobre o site </div>
<div id="intro"> Outro conteúdo </div>

Posso repetir um id? Clique aqui pra saber. IDs são como serial number de produtos, são usados para identificar um elemento único na página.
