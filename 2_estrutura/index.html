Parte 1: Termos

Estruturando o HTML

↑
↑

Todos os documentos HTML precisam ter uma estrutura obrifatória que inclui as seguites declarações e elementos:

<!DOCTYPE html>, <html>, <head>, and <body>

A primeira linha de todo documento HTML deve declarar o tipo de documento para que o browser saiba o que precisa ler. Se a versão for a última basta <!DOCTYPE html>, senão podemos declarar a versão e depois disso, abrimos a tag <html>.

↑
↑

<head> e <body>

↑
↑

Dentro do <html>, primeiro criamos o elemento <head> (cabeça). Tudo que vai dentro dessa tag não aparece na página propriamente dita. As informaçoes que são colocadas aqui são adicionais para a página: Título (que fica na barra do navegador), metadados que ajudam a encontrar a página e links de arquivos externos, como .css e .js, por exemplo.

Todo conteúdo que é visível na página está dentro do <body> (corpo). Isso inclui títulos, parágrafos, tabelas, imagens, menus e outros elementos.

↑
↑

Estrutura básica

↑
↑
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>Hello World</title>
</head>
<body>
<h1>Hello World</h1>
<p>This is a web page</p>
</body>
</html>

elementos
"Self-closing"

↑
↑

Nem todas as tags que são usadas precisam de encerramento, algumas delas tem todo conteúdo necessário apenas nos atributos ou nem precisam disso também. As mais comuns entre as self-closing são:

Dentro do <head>
<meta> <link>

Dentro do <body>
<br> <img> <hr> <source>
