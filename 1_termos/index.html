↑
↑

O HTML dá ao conteúdo do seu site estrutura e significado definindo esse conteúdo como cabeçalhos, parágrafos, menus, rodapés, images, links, etc...

Entendendo os termos

↑
↑

Enquanto estiver começando a trabalhar com HTML, pode ser que encontre alguns termos novos e estranhos. Com o tempo você vai acabar se tornando familiar com eles, então não se preocupe que vai ficando mais fácil depois 😊

Elemento

↑
↑

Elementos são designadores que definem a estrutura e conteúdo dos objetos na páginas. Alguns dos mais usados são headings, parágrafos, spans e divs. Elementos são identificados com tags

<a>
↑
↑

tags

↑
↑

O uso de < >, ao redor do elemento, define uma tag. Tags, geralmente, são compostas por um par de abertura e fechamento.

A abertura de tag marca o começo daquele elemento. Composta de [menor que] + [nome do elemento] + [maior que]

O fechamento de tag encerra esse mesmo elemento. É composto de [menor que] + [barra] + [nome do elemento] + [maior que]

Tudo que for escrito dentro dessas tags será o conteúdo dela, então, para criar uma tag de link (a), escreve-se:

<a> html </a>
↑
↑

atributos

↑
↑

Atributos são propriedades usadas para dar informações adicionais sobre o elemento. Os mais comuns são:

id: Identificam um elemento;
class: Classificando um elemento;
src: Procuram a fonte (source) que vai ser embutida no elemento;
href: Referem-se a hyperlinks locais ou externos.

<a id="link" class="menu" href="http://movile.com"> Movile </a>
